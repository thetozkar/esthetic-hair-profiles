# Esthetic Hair - Profil Oluşturucu

WhatsApp için profesyonel profil görselleri oluşturma sistemi.

## 🎯 Özellikler

- ✅ Kişiselleştirilmiş profil oluşturma (isim + rol + şube)
- ✅ 4 farklı şube desteği (Turkey, Miami, Mexico, Brasil)
- ✅ 20+ farklı rol seçeneği
- ✅ JPG olarak indirme
- ✅ Mobil uyumlu
- ✅ Hazır şablon profiller

## 🚀 Canlı Kullanım

Web sitesini ziyaret edin: [Buraya deploy linki gelecek]

## 💻 Yerel Kurulum (Geliştirici için)

```bash
# Bağımlılıkları yükle
pnpm install

# Geliştirme sunucusunu başlat
pnpm dev
```

## 📝 Nasıl Kullanılır

1. Sayfayı aç
2. İsim, şube ve rol seç
3. "Profil Oluştur" butonuna tıkla
4. "İndir JPG" ile indir
5. WhatsApp profil resmi olarak kullan
